# 🌊 Flood Relief Management System  

---

## 📌 Project Introduction

Sri Lanka frequently experiences flood situations that affect thousands of families each year.  
The **Flood Relief Management System** is a web-based application designed to support the **collection, management, and monitoring of flood relief requests** during such emergencies.

This system allows **affected individuals** to submit relief requests easily, while **administrators** can efficiently manage users and analyze relief data through summarized reports.  

---

## 🎯 Objectives of the System

- ✅ Allow affected people to register and request flood relief  
- ✅ Enable administrators to manage users and relief data  
- ✅ Provide summarized and filtered reports for decision-making  
- ✅ Ensure role-based access and data security  
- ✅ Maintain a simple, user-friendly interface  

---

## 👥 User Roles

### 👤 Affected Person (User)
- Register and login to the system  
- Submit flood relief requests  
- View, update, and delete their own requests  

### 🛠 Administrator (Admin)
- View all registered users  
- View detailed user information  
- Generate summarized reports based on filters  
- Delete users when required
